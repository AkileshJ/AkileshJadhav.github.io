Project 1: Mortality Analysis

A Mortality Analysis performed on Covid 19 data listing world wide entries.

June 8, 2024 Myself

For this project, the data fetched is a csv file of Covid 19 world wide report from ourworldindata.org listing all the entries of individuals that are infected, vaccinated, death count including vaccinated and non vaccinated individuals.

The data in this report list everyday entries of each individual world wide for the past 4 years from year 2020 to year 2024. I have segregated and isolated elements from this database to highlight local & global numbers, infection rate, effectiveness of vaccine, probability of contracting covid and if so its mortality rate.

I have completed this project in SQL Server Management studio. Below is the link to see the entire project and syntaxes used to perform analysis.
